# Dycetix System

## Overview
Two Django projects:
- **Customer** (Port 8000): Public website with forms, blog, static pages
- **Admin** (Port 8001): Management dashboard

## Quick Start
\\\ash
# Run customer site (in one terminal)
cd customer
python manage.py runserver 8000

# Run admin site (in another terminal)  
cd admin
python manage.py runserver 8001
\\\

## URLs
- Customer: http://localhost:8000
- Admin: http://localhost:8001/admin
